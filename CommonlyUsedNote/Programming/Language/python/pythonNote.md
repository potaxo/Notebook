## Mathematical operations
```python
import math
print(2 ** 3)  # Exponentiation
print(10 // 3)  # Floor division
print(math.sqrt(16))  # Square root
print(math.pi)  # Value of pi
print(math.sin(math.radians(90)))  # Value of sin(90)
```
